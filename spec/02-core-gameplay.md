# Core gameplay

## Core loop (шаблон)

Опиши в 3–7 шагов:

1. Вход в сессию (контекст)
2. Основное действие игрока
3. Награда/прогресс
4. Триггер следующей сессии

## Параметры сессии

- Длительность: целевой диапазон
- Сложность: как растёт
- Точки «микро-наград»

## UX требования

- Быстрый старт
- Ясная цель/фидбек
