# Overview

## Цель

Собрать «мастер-спеку» (единая точка правды) для попытки **yg4**: требования, research, план реализации.

## Платформа

- Unity: **2022.3 LTS**
- Target: **WebGL**
- Platform SDK: **Yandex Games (YG SDK)**

## Что в scope

- Определение core loop + meta progression
- Монетизация: rewarded / interstitial / IAP (RUB) + remove ads + starter pack
- Опционально: leaderboards
- Интеграция YG SDK (init/env/auth/lang + ads + IAP + saves + leaderboards)
