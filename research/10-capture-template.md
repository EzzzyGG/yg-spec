# Capture template (витрина / топ)

Заполняем по одной строке на игру/позицию в топе.

| Игра | URL | Платформа/источник | Жанр | Core mechanic (1 фраза) | Loop (30–60 сек) | Мета/прогресс | Монетизация (rewarded / interstitial / IAP) | Leaderboard | Заметки |
|---|---|---|---|---|---|---|---|---|---|
|  |  |  |  |  |  |  |  |  |  |

## Правила снятия

- **Core mechanic**: один глагол + объект (например: «сливать блоки», «стрелять по шарам», «тащить предметы», «соединять одинаковые»).
- **Loop**: что игрок делает *каждую минуту*.
- **Мета**: есть ли апгрейды/коллекции/миссии/карта/прокачка.
- **Монетизация**: если не видно — ставь `?` и помечай в заметках, что надо проверить в игре.
