# Research

Цель папки `research/` — быть **рабочей тетрадью**: источники → сбор кандидатов → кластеризация → shortlist-10.

## Поток работы

1. Открыть источники из [`00-sources.md`](./00-sources.md)
2. Заполнить первичный сбор кандидатов: [`20-candidates-raw.md`](./20-candidates-raw.md)
3. Кластеризовать: [`30-clusters.md`](./30-clusters.md)
4. Собрать shortlist-10: [`40-shortlist-10.md`](./40-shortlist-10.md)

## Шаблоны

- Как снимать/фиксировать данные: [`10-capture-template.md`](./10-capture-template.md)

## Правила

- Все ссылки фиксировать прямыми URL.
- Для каждой игры сохранять: название, URL, платформа-источник, жанр/механика, тип (endless/campaign), заметки.
- Shortlist-10: 5 endless + 5 campaign, чтобы потом проще балансировать пайплайн продакшена.
